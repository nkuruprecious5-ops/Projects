# To-do Task Board

A simple and responsive task board built with **HTML, CSS, and JavaScript**.

## Features
- Add tasks with title and optional description
- Mark tasks as **Done** or **Undo** back to To Do
- Edit and delete tasks
- Drag-and-drop between To Do and Done
- LocalStorage persistence (tasks stay after refresh)
- Mobile-friendly layout

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

## Project Structure
- `index.html` - main UI
- `styles.css` - styling and responsive layout
- `script.js` - task logic, rendering, and localStorage

## Run Locally
Because this project is static:

1. Open `index.html` directly in your browser, or
2. Use a local static server (recommended).

## Notes
- Data is saved in browser localStorage only.
- No backend or database is required.

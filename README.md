# Precious Projects Portfolio

This repository contains three web development projects by **Nkurunziza Precious**.

## Projects

### 1. To-do Task Board
A responsive task board built with HTML, CSS, and JavaScript.

**Highlights**
- Add, edit, and delete tasks
- Mark tasks as Done/Undo
- Drag and drop between task sections
- LocalStorage persistence

Project folder: [To-do Task Board](./To-do%20Task%20Board)

---

### 2. Antimicrobial Landing Page
A multi-page health awareness website with a modern hero-style design.

**Highlights**
- Home, About, Services, Gallery, Contact pages
- Responsive navigation with mobile menu
- Image sections and gallery cards
- Shared footer + custom favicon

Project folder: [Anti microbial Landing Page](./Anti%20microbial%20Landing%20Page)

---

### 3. Single-Page Contact Manager
A contact manager using PHP, MySQL, and AJAX for no-refresh interactions.

**Highlights**
- Add and delete contacts without page reload
- PHP API + MySQL database
- Validation and error messages
- Health check endpoint for debugging

Project folder: [Contact Manager](./Contact%20Manager)

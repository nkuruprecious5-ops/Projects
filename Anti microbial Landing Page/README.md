# Antimicrobial Landing Page

A multi-page awareness website designed around an antimicrobial health campaign theme.

## Pages
- `index.html` - Home page with hero section and image highlights
- `about.html` - Mission and focus
- `services.html` - Program and outreach services
- `gallery.html` - Image gallery section
- `contact.html` - Contact and partnership info

## Features
- Dark hero-style visual design inspired by your reference
- Responsive top navigation with mobile menu toggle
- Home image section with featured content
- Gallery with image cards and captions
- Shared footer across pages
- Custom favicon (`favicon.svg`)

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

## Project Structure
- `styles.css` - shared styling for all pages
- `script.js` - mobile nav toggle + footer year logic
- `favicon.svg` - site icon

## Run Locally
This is a static site.

1. Open `index.html` in your browser, or
2. Serve the folder with a static server for best navigation behavior.

## Notes
- External images are loaded via public URLs.
- You can replace those URLs with local image files anytime.
